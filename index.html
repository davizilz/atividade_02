10 tags de HTML

1.html


A tag HTML é a raiz de um documento HTML. Ela envolve todo o conteúdo da página.

-Demonstração Visual:A moldura de toda a tela.

2.head

A tag head contém informações sobre o documento, como o título, metadados e links para arquivos externos.

-Demonstração Visual: A parte superior da tela, onde ficam as informações sobre a página.

3.body

A tag body contém o conteúdo visível da página, como texto, imagens e outros elementos.

-Demonstração Visual: A parte principal da tela, onde fica o conteúdo principal da página.

4.h1

A tag h1 é usada para criar um título de nível 1, que é o título maior e mais importante em uma página HTML.

-Demonstração Visual

: O título mais importante da página, geralmente exibido em uma fonte maior e em negrito.

5.p

A tag p é usada para criar um parágrafo de texto em uma página HTML.

-Demonstração Visual

: Um bloco de texto, escrita em ordem e tamanho normal.

6.a

A tag a é usada para criar um link para outra página ou recurso na web.

-Demonstração Visual: Um link clicável que leva a outra página ou

7.img

A tag img é usada para inserir uma imagem em uma página HTML.

-Demonstração Visual: Uma imagem exibida na página.

8.div

A tag div é usada para juntar elementos HTML em uma divisão ou seção da página.

-Demonstração Visual

: Um bloco de elementos HTML agrupados juntos.
